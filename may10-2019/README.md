# Topothek Computer Vision Results

Results of Computer Vision processing of Topothek Data Assets

## Getting Started

The results are in CSV Format to be uploaded into a RDBMS database table. The structure of the CSV is as follows:

id: sequential number for managing purposes
language: language of the concept generated by computer vision
document_id: document_id from topothek's Digital Asset
concept: the actual concept generated by computer vision
